PROJECT: Movie Recommendation Systems

RECOMMENDATION SYSTEMS:
1) Recommendation systems are a collection of algorithms used to recommend items to users based on information taken from the user. 
2) These systems can be commonly seen in online stores, movies databases and job finders. 
3) Some of examples are -> Netflix, Amazon Prime

ABOUT PROJECT:
The project has two movie Recommendation Systems developed ->
1) Collaborative filtering based recommendation system
2) Content based recommendation system
